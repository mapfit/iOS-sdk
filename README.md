## Installation

#### CocoaPods
You can use [CocoaPods](http://cocoapods.org/) to install `Mapfit` by adding it to your `Podfile`:

```ruby
platform :ios, '8.0'
use_frameworks!
pod 'YourLibrary'
```

To get the full benefits import `YourLibrary` wherever you import UIKit

``` swift
import UIKit
import YourLibrary
```
